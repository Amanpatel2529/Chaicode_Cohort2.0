## How to Add Your Blog

1. Create or publish your post (Hashnode, personal blog, GitHub Pages).
2. Add an entry in this file under the appropriate section using the Template below.
3. Create a pull request (or notify a maintainer) so we can review and merge your link.

### Suggested fields for each entry

- **Title**: Full post title
- **Author**: Your name
- **Date**: YYYY-MM-DD (publish date)
- **Link**: Full URL to the post
- **Tags**: Comma-separated tags or topics
- **Summary**: 1–2 sentence summary
- **Source**: Hashnode/GitHub/Personal

## Template
Copy-paste this template when adding a new entry:

```
- Title: <Post Title> — <Author> (YYYY-MM-DD)
	- Link: <https://example.com/post>
	- Tags: #tag1, #tag2
	- Summary: Short one-line summary of the post.
	- Source: Hashnode | Personal | GitHub Pages
```
